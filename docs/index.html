Full rhwp surface

Uses a local rhwp-studio build with toolbar, document editing, table and cell selection, and HWP/HWPX export.

VS Code save lifecycle

Dirty state, native save, Save As, revert, and backup are routed through an editable CustomEditorProvider.

Format-aware writes

HWP files save as HWP, HWPX files save as HWPX, and mismatched bytes are rejected before disk writes.

Release discipline

Packaged with explicit smoke gates.

npm run release:local

The local release gate type-checks the extension and React WebView, builds the bundled rhwp runtime into resource/rhwp-studio, verifies HWP hardening assumptions, packages the VSIX, and checks that the VSIX contains runtime WASM assets while excluding samples and vendor sources.
