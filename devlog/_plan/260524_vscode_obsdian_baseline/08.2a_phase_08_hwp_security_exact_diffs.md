## 1. Local rhwp-studio Bundle Build Path

Chosen packaged path:

```text
resource/rhwp-studio/
```

Reason:

- `resource/` is already package-visible.
- `.vscodeignore` does not exclude `resource/**`.
- The extension can map it with `webview.asWebviewUri`.
- It avoids mixing third-party runtime assets into `out/webview`.


Path table:

```text
source:       vendor/rhwp-studio-dist/index.html
build output: resource/rhwp-studio/index.html
runtime URI:  webview.asWebviewUri(Uri.file(`${extensionPath}/resource/rhwp-studio/index.html`))
VSIX path:    extension/resource/rhwp-studio/index.html
```

must not contain `https:` or `*` for HWP runtime.

Write sequence pseudo-code:

```typescript
async function resolveHwpSaveTarget(sourceUri: Uri, ext: string): Promise<Uri | undefined> {
  if (ext === '.hwpx') {
    const suggested = Uri.file(join(dirname(sourceUri.fsPath), basename(sourceUri.fsPath, ext) + '.converted.hwp'));
    try {
      await workspace.fs.stat(suggested);
      return await window.showSaveDialog({ defaultUri: suggested, filters: { 'HWP': ['hwp'] } });
    } catch {
      return suggested;
    }
  }
  return sourceUri;
}

async function writeHwpAtomically(targetUri: Uri, bytes: Uint8Array): Promise<void> {
  const tempUri = targetUri.with({ path: `${targetUri.path}.tmp-${Date.now()}` });
  await workspace.fs.writeFile(tempUri, bytes);
  await workspace.fs.rename(tempUri, targetUri, { overwrite: false });
}
```

Validation:

```text
HWP save payload must start with OLE_MAGIC.
HWPX input may start with ZIP_MAGIC, but exported HWP save payload must be OLE_MAGIC.
Abort write and emit `hwp:saveResult` failure if max size or signature check fails.
```
