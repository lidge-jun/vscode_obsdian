# devlog/ Agent Rules

This directory stores active and completed work records for `vscode_obsdian`.

Rules:
- Active plans live under `devlog/_plan/YYMMDD_slug/`.
- Completed work summaries move to `devlog/_fin/YYMMDD_slug/`.
- Phase documents must use lexicographic numeric prefixes: `00_overview.md`, `01_phase_01_slug.md`, `02_phase_02_slug.md`.
- Research appendices use `90_` or higher prefixes.
- Do not create bare `PLAN.md`, `DIFF_PLAN.md`, `PHASES.md`, `RCA.md`, or `plan.md`.
- Keep this file in English.
